What makes up the Node.js platform?

Modules make up the platform. This is a list from the manual.

5 Line Web Server

	
			// use Node.js http module by using module system
			var http = require('http');
			http.createServer(function (request, response) {
  				response.writeHead(200, {'Content-Type': 'text/plain'});
  				response.end('Hello World\n');
			}).listen(8124);
			console.log('Server running at http://127.0.0.1:8124/');

Can I get other modules?

  • I need to parse a CSV, store data in to a MongoDB, and send email. How am I supposed to use modules?
    1. Search npmjs.org for the module you need
    2. npm install {{csv}} --save
    3. NPM downloads your module from the NPM registry
    4. Because you called the --save this module and its version are recorded in a package.json file

    Tip: Choose your modules wisely. Check features and github.
